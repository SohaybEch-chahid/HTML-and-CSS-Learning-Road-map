# HTML & CSS Learning Roadmap

## Introduction

This repository contains my complete journey of learning HTML and CSS from scratch. The goal is to build a strong foundation in web development before moving on to JavaScript, frameworks, and backend technologies.

---

# What You Need Before Learning HTML & CSS

You do not need extensive programming knowledge before starting HTML and CSS.

Helpful skills include:

- Basic computer knowledge
- Creating files and folders
- Using a text editor
- Using a web browser
- Basic terminal usage
- Git and GitHub (optional but recommended)

---

# Understanding How Websites Work

Before learning HTML and CSS, understand the basic flow of a website:

```text
Browser (Client) <----> Server
```

Examples:

- You enter a URL.
- The browser sends a request.
- The server sends back HTML, CSS, JavaScript, images, and other resources.
- The browser renders the webpage.

---

# The Roles of HTML, CSS, and JavaScript

Think of a website like a house.

| Technology | Purpose       |
| ---------- | ------------- |
| HTML       | Structure     |
| CSS        | Design        |
| JavaScript | Functionality |

### Example

HTML:

```html
<button>Click Me</button>
```

CSS:

```css
button {
  background: blue;
  color: white;
}
```

JavaScript:

```javascript
button.onclick = function () {
  alert("Hello");
};
```

# Browser Developer Tools

Learn to use browser developer tools.

Open them using:

```text
F12
```

Important tabs:

- Elements
- Inspector
- Console
- Network

Developer tools are essential for debugging and understanding webpages.

# Things You Do Not Need Yet

Avoid learning these before mastering HTML and CSS:

- React
- Next.js
- Tailwind CSS
- Bootstrap
- TypeScript
- Node.js
- PHP
- Databases

Focus only on:

```text
HTML
↓
CSS
↓
Responsive Design
↓
JavaScript
↓
GitHub Pages
↓
React
```

# Best Learning Method

For every topic:

1. Learn the concept.
2. Write the code yourself.
3. Build a mini project.
4. Push it to GitHub.

Avoid spending too much time watching tutorials without practicing.
